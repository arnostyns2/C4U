Observatie:
- Mobile layout voelt perfect aan
- Desktop layout voelde onrustig bij webshop-card

Oorzaak:
- Op desktop wordt de webshop-card breder (grid-column: span 2)
- Background-image gebruikte background-size: cover
- Hierdoor werd webshop.png onbedoeld gecropped

Oplossing:
- Mobile-first blijft onveranderd (background-size: cover)
- Desktop-only override:
  - Webshop-card gebruikt background-size: contain
  - Andere kaarten blijven cover
  - Visuele dominantie blijft via breedte, niet via crop

Waarom dit correct is:
- Screenshots en UI-images werken beter met contain
- Inspiratiebeelden werken beter met cover
- Funnel blijft logisch en rustig

Belangrijk:
- Nooit height aanpassen om dit te fixen
- Nooit images stretchen
- Altijd via media queries oplossen
