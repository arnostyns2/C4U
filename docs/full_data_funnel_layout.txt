Funnel volgorde:

Layout principes:
- Mobile-first
- Desktop optimalisatie via media queries
- Consistente card heights
- Aspect-ratio altijd gerespecteerd

Afbeeldingen:
- Inspiratiebeelden → background-size: cover
- UI / webshop screenshots → background-size: contain (desktop only)

Dominantie:
- Webshop is dominant door grid-span
- Niet door hoogte
- Niet door schreeuwerige kleuren

Waarom dit schaalbaar is:
- Nieuwe links kunnen makkelijk toegevoegd worden
- A/B testen mogelijk via volgorde
- Analytics events zijn consistent

Belangrijk voor toekomst:
- Nieuwe webshop screenshots altijd testen op desktop
- Mobile blijft leidend voor eerste indruk
